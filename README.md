- 🎤 **Áudio ambiente** com volume máximo a cada 10 segundos

- 📊 **Painel Web** com layout horizontal e controles
- 🤖 **Integração com Telegram** (envio automático)
- 📄 **Exportação em PDF** com relatório completo
- 🔒 **Login protegido** (admin/SpyWatdon3609)
- 🕵️ **Modo escondido** (pasta oculta)
